Portafolio UX/UI

Violeta Alanoca. Diseñadora en Comunicación Visual.

SERVICIOS

UX

Mi enfoque en UX (Experiencia de Usuario) se centra en la investigación y diseño para crear experiencias digitales intuitivas y satisfactorias. Mi objetivo es comprender las necesidades y comportamientos de los usuarios para diseñar soluciones que faciliten y mejoren su interacción con productos y servicios.

Herramientas utilizadas: Trello, SimilarWeb, HotJar, Maze, Lyssna, Google Forms, Typeform, Optimal Workshop, entre otras.

UI

En UI (Interfaz de Usuario), me dedico a diseñar interfaces visuales atractivas y coherentes que faciliten la interacción del usuario con el producto digital. Trabajo en la estética del diseño, asegurándome de que los elementos visuales sean intuitivos y agradables.

Herramientas utilizadas: Figma.

FRONT-END

Además del diseño, tengo habilidades en desarrollo front-end, donde uso HTML, CSS y JavaScript para convertir mis diseños en páginas web funcionales. Estas tecnologías me permiten dar vida a mis diseños y asegurar que se vean bien y funcionen perfectamente en todos los dispositivos.

Herramientas utilizadas: Visual Studio Code, Github, Html, Css, Javascript, etc.

SOBRE MÍ

Como diseñadora quiero proponer y desarollar soluciones que no solo enriquezcan visualmente los proyectos sino que puedan mejorar la experiencia del usuario de manera sencilla, coherente y creativamente. Siempre disponible para aprender, enseñar y colaborar con quiénes deseen trabajar en conjunto, especialmente si se trata de ser parte de proyectos de carácter tecnocreativo.
Soy una persona curiosa, con bastante capacidad de autoaprendizaje sobre todo en el campo de las artes mediales. Disfruto mucho experimentar con herramientas que me permitan expresar y crear nuevos imaginarios. Considero que es sumamente importante el cruce entre diversas disciplinas, tanto análogas como digitales. Entre mis varios hobbies se encuentran la programación creativa, el live-coding, el vj y mapping, la creación de filtros de Realidad Aumentada además del collage y la animación experimental.

PROYECTOS

Caso de estudio:

Tech Future 2025

Se llevó a cabo una propuesta de Diseño UX/UI para Tech Future, conferencia sobre las últimas tendencias tecnológicas. Bajo el objetivo de incentivar las inscripciones al evento se dió con un sitio web de fácil acceso y que considerara el interés del público objetivo. Este trabajo fue el examen final para la certificación de talento digital del Bootcamp de Diseño UX/UI.

VER MÁS
Caso de estudio:

Cruz Roja Chilena

Junto a Agencia Fusión trabajamos en una propuesta de UX/UI para el sitio web actual de Cruz Roja Chilena con el propósito de incentivar sus objetivos actuales, la donación monetaria y la llegada de nuevos voluntarios. A través de la metodología Design Thinking dimos con un producto óptimo para que el usuario pudieras navegar y accionar fácilmente a través de una landing page.

VER MÁS
Caso de estudio:

QQBB.cl

Junto a Agencia Fusión y bajo la metodología de Diseño Centrado en el Usuario desarrollamos un análisis del sitio web qqbb.cl, plataforma para la venta y compra de prendas recicladas para bebés. Se realizaron diversas pruebas a usuarios para dar con una nueva arquitectura de información la cual fue puesta prueba a través de prototipos en figma.
